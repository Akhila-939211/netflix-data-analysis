# 🎬 Netflix Data Analysis Project

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix dataset using Python.  
It helps to understand Netflix content trends such as movies vs TV shows, popular countries, genres, and yearly content growth.

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📂 Dataset
Netflix dataset contains information like:
- Show ID
- Type (Movie/TV Show)
- Title
- Director
- Cast
- Country
- Date Added
- Rating
- Duration
- Genres

---

## 📊 Analysis Performed

### 1. Movies vs TV Shows
Visualizes distribution of Movies and TV Shows on Netflix.

### 2. Top 10 Countries
Shows which countries produce the most Netflix content.

### 3. Top 10 Genres
Displays most common genres available on Netflix.

### 4. Content Added Over Years
Shows how Netflix content has grown over the years.

---

## 📈 Output
All graphs are generated using Matplotlib and Seaborn:
- Movies vs TV Shows chart
- Top Countries bar chart
- Top Genres bar chart
- Year-wise content trend
